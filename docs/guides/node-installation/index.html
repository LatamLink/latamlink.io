Installing LACChain EOSIO Nodes

Testnet Information#

EOSIO versionLACChain EOSIO
Chain ID2c1f36d2e3774cba3e47804b6463c207544ac24183194e0b96ffad31e8f4acd5
Genesis filegenesis.json

See our GitHub repository for more information about nodes in LACChain EOSIO.

1. Create an account#

In order to deploy nodes in the LACChain EOSIO network, it is important to have an account registered as a Partner or Non-Partner.

See the guide to create an account according to your user role:

  • Non-partner
  • Partner

2. Authenticate on the Dashboard#

The LACChain EOSIO network has a dashboard, which allows users to manage within the network, including the option to deploy various types of nodes.

Login with your Partner or Non-Partner account. See our tutorial on how to manage your private keys

3. Deploy a node#

Once authenticated with our account, select the administration option within the dashboard tools. In this section you will be shown the types of nodes that you can deploy depending on your type of account.

In order to create the node, we simply fill in the requested data in the form and execute the transaction.

Configuration examples#

The dashboard also has a section in which we can download an example of the configuration of the different types of node, we only have to access the option of node config within the tools.

4. Delete a node#

Inside the dashboard, there is also an option to be able to delete the nodes, simply by typing the name of the node that we want to delete.
